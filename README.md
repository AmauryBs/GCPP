# Gestion des commandes pour projets pédagogiques
## Introduction
Site de gestion de commandes pour des projets pédagogiques.

## Outils utilisés
Ce projet a été fait nativement et en partant de zéro :
* SQL (PostgreSQL 11)
* PHP 7.3
* HTML (avec balises structurantes)
* CSS
